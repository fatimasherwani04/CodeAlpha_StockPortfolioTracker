# CodeAlpha_StockPortfolioTracker
A beginner-friendly Python application that calculates the total investment value of a stock portfolio using predefined stock prices and user-input quantities.
A beginner Python application that calculates the total investment value of a stock portfolio.

## Description
The user enters stock symbols and quantities. The program uses predefined stock prices and calculates the total investment amount.

## Features
- Hardcoded stock prices
- User inputs stock symbols and quantities
- Calculates individual investment values
- Calculates total portfolio value
- Optionally saves results into a text or CSV file

## Technologies Used
- Python
- Dictionaries
- Loops
- File Handling
- Basic Arithmetic

## Concepts Practiced
- Dictionaries
- `while` loops
- `if-else`
- User input/output
- File handling
- Arithmetic operations

## How to Run

1. Download the project.
2. Open Terminal or Command Prompt.
3. Run:

```bash
python stock_portfolio_tracker.py
```
